关于mongoose数据库的安装和使用

今天试着使用了mongoose数据库，因为以前用过SQLserver，所以觉得不太难，然而还是撞到了几个坑，这里写一下

首先，怎么绝大多数教程都没有说在使用 npm install mongoose 这条命令之前要先下载mongodb？！！！！

当真是坑的我好惨啊，重要的事情说三遍:要先下载mongodb！要先下载mongodb！要先下载mongodb！

关于安装，如果在mac上安装了brew的情况下，可以直接执行命令

brew install mongodb

这个过程可能很快也可能很慢，我看网上不少人说几分钟就装好了，然而…我装了接近两个小时，我一度以为是进程挂掉了，所以请小伙伴们耐心等待

##关于使用

这里就不说概念性的东西了，那种东西网上的教材论打算，自己去找

###说一下具体怎么用：
要启动mongoDB，先打开两个命令行
第一个终端窗口输入　sudo mongod –config /usr/local/etc/mongod.conf
然后在第二个终端窗口输入　　mongo
然后在第二个终端窗口进行操作

如果你是把操作内容写在了js文件中，那么在启动mongoDB后可以node [filesName].js
